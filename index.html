Bandicoot toolbox

This demo from the MIT Media Lab shows how metadata can be safely used at large-scale through openPDS/SafeAnswers (http://openpds.media.mit.edu). The data is collected using the FUNF framework (http://funf.org) and the metadata processing through the bandicoot toolbox (http://bandicoot.mit.edu).

This application will collect mobile phone metadata (call and text logs with first name of the contact) and upload them to a secure MIT server. The data can then be displayed by entering the unique pin onto the demo screen. The data will only be used for display purposes and will be deleted one week after has been uploaded.
